# Data

- `DB1_PRISM_Weather_Daily_1981_2023.xlsx` — Daily weather (5 sites, 1981–2023)
- `DB2_Phenology.xlsx` — Budbreak, flowering, veraison, harvest dates
- `DB3_Yield_Harvest_date.xlsx` — Yield (tons/ha) and harvest date (DOY)
- `DB4_Fruit_composition.xlsx` — 12 grape analytes (2017–2023)
